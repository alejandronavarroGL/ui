# Alerts
Display an alert with text to users.

[Github repo](https://github.com/unicoderns/ui/tree/master/packages/alert)

```html
    <ui-alert :="args" :show="toggle" @show="show" @close="close" @hide="hide">{{ args.default }}</ui-alert>
```

## Accessibility
Use the following properties to change the default accessibility text

| Name | Property | Default |
| ---- | ---- | ----------- |
| role | aria:role | `alert` |
| buttonClose | aria:buttonClose | `close` |
